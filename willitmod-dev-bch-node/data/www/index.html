Bitcoin Cash Node (BCHN)

This app runs a Bitcoin Cash full node. Pruning is enabled by default.

RPC credentials

  • Username: bch
  • Password: your app password (Umbrel: "Show default credentials")

Ports

  • P2P: 28333
  • RPC: 28332
  • ZMQ (hashblock): 28334

Testnet / regtest

Edit data/node/bitcoin.conf and uncomment testnet=1 or regtest=1, then restart.
